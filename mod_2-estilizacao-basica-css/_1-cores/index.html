Entendendo cores no CSS

Cores Pré-Definidas e Current Color

Em CSS, há uma lista de cores pré-definidas que podem ser usadas diretamente nos estilos sem a necessidade de especificar valores em hexadecimal, RGB, ou HSL. Essas cores são padronizadas pela especificação CSS e são fáceis de lembrar e utilizar.

  • Cores pré-definidas

                
    // Exemplo com cores pré-definidas:
    
            
              <h1 id="pre-definidas">Cores pré-definidas</h1>
            
            
    // Aplicando seletores
    
            
              #pre-definidas {
                color: darkviolet;
              }

  • Current Color

                
    // Exemplo com currentcolor:
    
            
              <h1 id="current-color">Palavra-chave "currentcolor"</h1>
            
            
    // Aplicando seletores
    
            
              #current-color {
                color: orange;
                border: 1px solid currentcolor;
              }

RGB e RGBA

As funções RGB e RGBA em CSS são usadas para definir cores com base nos valores de vermelho (Red), verde (Green), azul (Blue) e, no caso do RGBA, também o valor de transparência (Alpha). Essas funções permitem um controle mais preciso sobre as cores e a opacidade do elemento.

  • RGB: A função rgb() define uma cor especificando os valores de vermelho, verde e azul. Cada valor pode variar de 0 a 255.

                
    // Exemplo com cores RGB:
    
            
              <h1 id="rgb">Cores RGB</h1>
            
            
    // Aplicando seletores
    
            
              #rgb {
                color: rgb(0, 0, 255);  // rgb(red, green, blue)
              }

  • RGBA: A função rgba() é semelhante à rgb(), mas inclui um quarto parâmetro, o canal alpha, que define a opacidade da cor. O valor de alpha varia de 0 (completamente transparente) a 1 (completamente opaco).

                
    // Exemplo com RGBA:
    
            
              <h1 id="rgba">Cores RGBA</h1>
            
            
    // Aplicando seletores
    
            
              #rgba {
                color: rgba(0, 255, 0, 0.8);  // rgb(red, green, blue, transparent(0.0 até 1.0))
              }

Hexadecimal e Hexadecimal com transparência

Em CSS, as cores podem ser definidas utilizando valores hexadecimais, que são representações numéricas da cor baseadas em componentes de vermelho, verde e azul (RGB). Também é possível adicionar um valor de transparência (alfa) ao formato hexadecimal.

  • Cores Hexadecimal: Um valor hexadecimal é composto por três pares de dígitos que representam os níveis de vermelho, verde e azul, com valores variando de 00 (ausência de cor) a FF (máxima intensidade de cor).

                
    // Exemplo com cores Hexadecimal:
    
            
              <h1 id="hex">Cores Hexadecimal</h1>
            
            
    // Aplicando seletores
    
            
              #hex {
                color: #FF0000; // #red 
              }
              #hex {
                color: #00FF00; // #green 
              }
              #hex {
                color: #0000FF; // #blue 
              }

  • Hexadecimal com transparência: O formato hexadecimal com transparência inclui um quarto par de dígitos para definir o nível de opacidade (alfa). O valor de opacidade varia de 00 (completamente transparente) a FF (completamente opaco).

                
    // Exemplo com Hexadecimal com transparência:
    
            
              <h1 id="hex-transparencia">Cores Hexadecimal com transparência</h1>
            
            
    // Aplicando seletores
    
            
              #hex-transparencia {
                color: #FF000000; // #red transparent
              }
              #hex-transparencia {
                color: #00FF00FF; // #green
              }
              #hex-transparencia {
                color: #0000FF00; // #blue transparent
              }

HSL e HSLA

HSL e HSLA são formas de representar cores em CSS baseadas em matiz (hue), saturação (saturation) e luminosidade (lightness). HSLA é uma extensão do HSL que inclui o canal de transparência (alpha).

  • Cores HSL: A função hsl() define cores de acordo com três parâmetros, são eles.

    • Matiz (Hue): Um valor em graus (de 0 a 360), representando a cor no círculo cromático (0 = vermelho, 120 = verde, 240 = azul).
    • Saturação (Saturation): Um valor em porcentagem (de 0% a 100%) que indica a intensidade da cor (0% = cinza, 100% = a cor mais viva possível).
    • Luminosidade (Lightness): Um valor em porcentagem (de 0% a 100%) que representa a claridade da cor (0% = preto, 100% = branco).
                
    // Exemplo com cores HSL:
    
            
              <h1 id="hsl">Cores HSL</h1>
            
            
    // Aplicando seletores
    
            
              #hsl {
                color: #FF0000;  
              }
              #hsl {
                color: #00FF00; 
              }
              #hsl {
                color: #0000FF; 
              }

  • Cores HSLA

                
    // Exemplo com HSLA:
    
            
              <h1 id="hsla">Cores HSLA</h1>
            
            
    // Aplicando seletores
    
            
              #hsla {
                color: #FF000000; 
              }
              #hsla {
                color: #00FF00FF; 
              }
              #hsla {
                color: #0000FF00; 
              }
