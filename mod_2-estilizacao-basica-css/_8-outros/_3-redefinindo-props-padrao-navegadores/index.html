Resetando as propriedades padrões dos navegadores

O reset de CSS é uma folha de estilos que irá sobrepor as propriedades de estilo CSS que os navegadores colocam por padrão nos elementos, ajudando com que os sites tenham o mesmo design em todos os navegadores.

          
Reset de configurações por arquivo reset.css

          <head>
            <link rel="stylesheet" href="reset.css">
            <link rel="stylesheet" href="style.css">
          </head>

          
Reset de algumas configurações padrões no CSS

          * {
            margin: 0;
            padding: 0;
            vertical-align: baseline;
            list-style: none;
            border: 0;
          }
