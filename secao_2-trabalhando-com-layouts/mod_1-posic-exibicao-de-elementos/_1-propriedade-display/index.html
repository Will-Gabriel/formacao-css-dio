Todo elemento possui um display?

Todo elemento HTML já vem com um valor de display como padrão

Cada elemento HTML já possui, por padrão, um valor da propriedade display já definidas. A tag <div>, por exemplo, possui um display em bloco, enquanto a tag <span> possui um valor inline como padrão.
