Entendendo Seletores CSS

Seletor por Tipo: Busca elementos por uma tag especifíca.

                
// Exemplo:

                h1 {
                background-color: blue;
                color: white;
                }

Seletor por ID (#): Busca elementos através do atributo "id".

                
// Exemplo:

                <div id="container">Essa div será manipulada no CSS através do seu ID</div>

                #container {
                background-color: blue;
                color: white;
                width: 1200px;
                }

Seletor por Classe: Busca elementos através do atributo "class".

                
// Exemplo:

                <div class="cards">Essa div será manipulada no CSS através do sua class</div>

                .cards {
                background-color: blue;
                color: white;
                width: 200px;
                }

Seletor Universal (*): Seleciona todos os elementos presentes em uma página HTML.

                
// Exemplo:

                * {
                margin: 0;
                padding: 0;
                font-family: "Poppins", sans-serif;
                }

Seletor de Atributo ([atrib]): Seleciona elementos que possuem um atributo especifíco em nosso documento HTML e conseguimos também buscar atributos com um valor especifíco.

                
// Exemplo:

                <ul>
                <li>
                    <a title="Prime" href="">Prime</a>
                </li>
                <li>
                    <a title="Spotify" href="">Spotify</a>
                </li>
                </ul>

                [title] {
                background-color: blue;
                color: white;
                }

                [title="Prime"] {
                background-color: red;
                color: white;
                }

                [href^="https://"] {
                background-color: red;
                color: white;
                }

                [href$="login"] {
                background-color: red;
                color: white;
                }

                [href*="login"] {
                background-color: red;
                color: white;
                }
