Entendendo Combinadores CSS

Agrupamento de seletor

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma vírgula (,)

Assim, não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazer um só bloco de CSS e reutilizar as mesmas regras!

          
// Exemplo: [SELETOR] , [SELETOR]

          
            <p class="texto">Lorem ipsum dolor sit amet.</p>
            <a id="link-referencia" href="#">Clique para ver a documentação</a>
            <a id="link-suporte" href="#">Clique para entrar em contato</a>
            <a id="link-voltar" href="#">voltar</a>

            <button>Fechar página</button>
            <button id="botao-fechar">Fechar página</button>
          
          
// Aplicando seletores

          
            p.texto,
            #link-referencia,
            #link-suporte,
            #link-voltar,
            [id^="botao"], button {
              background-color: blue;
              color: white;
            }

Combinador Descendente

Um combinador descendente é um conceito dos seletores em CSS. Ele é usado para combinar seletores de maneira a especificar elementos filhos diretos ou indiretos de um determinado elemento.

          
// Exemplo: [SELETOR] [COMBINADOR] [SELETOR]

          
            <ul>
              <li class="fundo-verde" id="lista-01">
                <div>Item 1</div>
                <ul id="sublista-01">
                  <li>Subitem A</li>
                  <li>Subitem B</li>
                </ul>
              </li>

              <li class="fundo-verde" id="lista-02">
                <div>Item 2</div>
                <ul id="sublista-02">
                  <li>Subitem A</li>
                  <li>Subitem B</li>
                </ul>
              </li>

              <li class="fundo-azul" id="lista-03">
                <div>Item 3</div>
                <ul id="sublista-03">
                  <li>Subitem A</li>
                  <li>Subitem B</li>
                </ul>
              </li>
            </ul>
          
          
// Aplicando seletores

          
            li li {
              color: green;
            }

            #lista-01 li {
              background: red;
            }

            .fundo-verde li {
              background: green;
            }

            .fundo-azul li {
              background: blue;
            }

            .fundo-verde #sublista-01 li {
              background: black;
            }

Combinador Filho

O combinador de filhos em CSS é utilizado para selecionar elementos que são filhos diretos de um outro elemento.

          
// Exemplo: [SELETOR] [COMBINADOR] [SELETOR]

          
            <div class="fundo-verde">
              <p>Parágrafo 1 - dentro da div</p>
              <p>Parágrafo 2 - dentro da div</p>
              <span>
                <p>Parágrafo 3 - dentro da div, dentro do span</p>
              </span>
            </div>

            <div class="fundo-verde">
              <p>Parágrafo 4 - dentro da div>
              <p>Parágrafo 5 - dentro da div>
              <span>
                <p>Parágrafo 6 - dentro da div, dentro do span</p>
              </span>
            </div>
                
            <div class="fundo-azul">
              <p>Parágrafo 7 - dentro da div</p>
              <p>Parágrafo 8 - dentro da div</p>
              <span>
                <p>Parágrafo 9 - dentro da div, dentro do span</p>
              </span>
            </div>

            <p>Parágrafo 10 - fora da div>
            <p>Parágrafo 11 - fora da div>
          
          
// Aplicando seletores

          
            div > p {
              background: lightblue;
            }

            .fundo-azul > p {
              background: lightgreen;
            }

            .fundo-verde > p {
              background: lightred;
            }

Combinador Irmão

O combinador de irmãos em CSS é usado para selecionar elementos que são irmãos de outro elemento, ou seja, que compartilham o mesmo elemento pai.

          
// Exemplo 1: [SELETOR] [COMBINADOR] [SELETOR]

          
            <div class="fundo-verde">
              <p>Parágrafo 1 - dentro da div</p>
              <p>Parágrafo 2 - dentro da div</p>
              <span>
                <p>Parágrafo 3 - dentro da div, dentro do span</p>
              </span>
            </div>
            
            <p>Parágrafo 4 - fora da div>
            <p>Parágrafo 5 - fora da div>

            <div class="fundo-verde">
              <p>Parágrafo 6 - dentro da div</p>
              <p>Parágrafo 7 - dentro da div</p>
              <span>
                <p>Parágrafo 8 - dentro da div, dentro do span</p>
              </span>
            </div>
                
            <p>Parágrafo 9 - fora da div>
            <p>Parágrafo 10 - fora da div>

            <div class="fundo-azul">
              <p>Parágrafo 11 - dentro da div>
              <p>Parágrafo 12 - dentro da div>
              <span>
                <p>Parágrafo 13 - dentro da div, dentro do span</p>
              </span>
            </div>

            <p>Parágrafo 14 - fora da div>
            <p>Parágrafo 15 - fora da div>
          
          
// Aplicando seletores

          
            --------------------------------
            --- Combinador adjacente (+) ---
            --------------------------------
            div + p {
              color: tomato;
            }

            .fundo-verde + p {
              background: lightgreen;
            }

            -------------------------------------
            --- Combinador irmão em geral (~) ---
            -------------------------------------
            div ~ p {
              color: tomato;
            }

            .fundo-verde ~ p {
              background: lightgreen;
            }
          
        
          
// Exemplo 2: [SELETOR] [COMBINADOR] [SELETOR]

          
            <div class="fundo-verde">
              <p>Elemento 01</p>
              <p>Elemento 02</p>
              <span>
                <p>Elemento 03</p>
                <p>Elemento 04</p>
              </span>
              <p>Elemento 05</p>
              <p>Elemento 06</p>
            </div>
          
          
// Aplicando seletores

          
            --------------------------------
            --- Combinador adjacente (+) ---
            --------------------------------
            .fundo-verde span p {
              background: lightgreen;
            }

            .fundo-verde span + p {
              background: lightgreen;
            }

            -------------------------------------
            --- Combinador irmão em geral (~) ---
            -------------------------------------
            .fundo-verde span ~ p {
              background: lightgreen;
            }

            .fundo-verde p ~ p {
              background: lightgreen;
            }
