Entendendo Dimensionamento e Espaçamento CSS

Largura e Altura

Em CSS, as propriedades largura (width) e altura (height) são usadas para definir o tamanho dos elementos, controlando sua dimensão horizontal e vertical, respectivamente.

          
// Exemplo:

          
            <div>Trilha de CSS</div>

            <div id="div-01">
              Sou a div 01
              <div id="div-02">
                Sou a div 02
              </div>
            </div>
          
          
// Aplicando seletores

          
            div {
              width: 200px;
              height: 100px;
              background: lightblue;
            }

            #div-01 {
              width: 200px;
              height: 300px;
              background: lightblue;
            }
            #div-02 {
              background: lightgreen;
              width: inherit;
            }

Largura e Altura Máxima e Mínima

Em CSS, além de definir a largura e altura de um elemento, você pode controlar os limites usando as propriedades de largura máxima (max-width), largura mínima (min-width), altura máxima (max-height) e altura mínima (min-height). Isso permite um controle mais flexível sobre o dimensionamento dos elementos, especialmente em layouts responsivos.

          
// Exemplo:

          
            <button id="botao-01">Fechar página</button>
            <button id="botao-02">Fechar página</button>
          
          
// Aplicando seletores

          
            #botao-01 {
              min-width: 200px;
              min-height: 100px;
              background: lightblue;
            }

            #botao-02 {
              min-width: 200px;
              max-width: 300px;
              min-height: 100px;
              max-height: 200px;
              background: lightblue;
            }

Margin

A propriedade margin em CSS é usada para criar espaço em torno de elementos, fora de suas bordas. Ela controla a distância entre um elemento e seus elementos vizinhos, tanto no sentido vertical quanto horizontal.

          
// Exemplo:

          
            <div id="elemento-01"></div>
            <div id="elemento-02"></div>
          
          
// Aplicando seletores

          
            div {
              width: 200px;
              height: 200px;
            }

            #elemento-01 {
              background: orange;
              margin-bottom: 50px; 
            }
            #elemento-01 {
              background: purple;
              margin: 20px 10px;  // aplica 20px em cima e baixo, e 10px nas laterais
            }
            #elemento-01 {
              background: purple;
              margin: 20px 10px 40px; // 20px em cima, 10px nas laterais e 40px na parte de baixo
            }
            #elemento-01 {
              background: purple;
              margin: 20px 10px 20px 30px;  // 20px(cima), 10px(direita), 20px(baixo) e 30px(esquerda)
            }

            #elemento-02 {
              background: purple;
              margin-top: 10px; 
              margin-left: 25px;
            }
            #elemento-02 {
              background: purple;
              margin: 30px; // aplica 30px em todos os lados
            }
            #elemento-02 {
              background: purple;
              margin: auto; // centraliza a div aplicando margins iguais nas laterais
            }
            #elemento-02 {
              background: purple; 
              margin: 20px auto; // 20px(cima e baixo), centraliza a div aplicando margins iguais nas laterais
            }

Padding

A propriedade padding em CSS define o espaçamento interno de um elemento, ou seja, o espaço entre o conteúdo do elemento e suas bordas. Ao contrário do margin, que cria espaço externo ao redor do elemento, o padding afeta o espaço dentro do próprio elemento.

A aplicação dos valores funcionam da mesma forma que o margin, podendo aplicar valores especificos para cada lado (cima, baixo, esquerda e direita).

          
// Exemplo:

          
            <div id="elemento-01">Elemento 01 da div 01</div>
            <div id="elemento-02">Elemento 02 da div 02</div>
          
          
// Aplicando seletores

          
            div {
              width: 200px;
              height: 200px;
            }

            #elemento-01 {
              background: orange;
              padding: 20px;
            }

Box Sizing

A propriedade box-sizing em CSS define como o navegador deve calcular a largura e a altura de um elemento, considerando o padding e a borda.

  • content-box (padrão): A largura e altura definidas se aplicam apenas ao conteúdo do elemento. O padding e a borda são adicionados além dessas dimensões.
  • border-box: A largura e altura incluem o conteúdo, o padding, e a borda. Isso significa que o tamanho total do elemento não excede o valor definido, e o conteúdo será ajustado para caber dentro do espaço disponível.
          
// Exemplo:

          
            <div id="elemento-01">Elemento 01 da div 01</div>
            <div id="elemento-02">Elemento 02 da div 02</div>
          
          
// Aplicando seletores

          
            div {
              width: 200px;
              height: 200px;
            }

            ---------------------
            ---- content-box ----
            ---------------------
            #elemento-01 {
              width: 200px;
              padding: 20px;
              border: 10px solid black;
              box-sizing: content-box;   // Padrão aumenta o tamanho total -> 200px + 20px + 20px = 240px
            }

            --------------------
            ---- border-box ----
            --------------------
            #elemento-02 {
              width: 200px;
              padding: 20px;
              border: 10px solid black;
              box-sizing: border-box;   // width não muda o tamanho total de 200px 
            }
