Entendendo Imagens no CSS

Propriedade Object-Fit

A propriedade object-fit no CSS é usada para controlar como o conteúdo de um elemento substituível, como imagens ou vídeos, se ajusta dentro de um contêiner. Aqui estão as opções de ajuste:

  • fill: O conteúdo é esticado para preencher todo o espaço do contêiner, o que pode distorcer a imagem.

  • contain: Mantém a proporção da imagem, ajustando-a para caber no contêiner. Pode deixar áreas vazias ao redor se o aspecto da imagem não corresponder ao do contêiner.

  • cover: Mantém a proporção da imagem, mas a ajusta para cobrir todo o contêiner, o que pode resultar no corte da imagem.

  • none: A imagem é exibida em seu tamanho original, sem redimensionamento.

  • scale-down: A imagem será redimensionada como se none ou contain fosse aplicado, escolhendo o menor tamanho possível sem distorção.

Essa propriedade é muito útil para evitar a distorção de imagens em layouts responsivos, onde as dimensões do contêiner podem mudar com o tamanho da tela (DevDoc) (HubSpot Blog) (SitePoint).

Propriedade Object-Position

A propriedade object-position no CSS é usada para controlar o posicionamento do conteúdo substituível (como imagens ou vídeos) dentro do contêiner, especialmente quando o conteúdo foi redimensionado pela propriedade object-fit. Por padrão, o conteúdo é centralizado dentro do contêiner, mas você pode usar object-position para ajustar a posição.

  • Palavras-chave: Você pode usar palavras como top, bottom, left, right e center para posicionar a imagem dentro do contêiner. Por exemplo, object-position: top right; posicionaria o conteúdo no canto superior direito.

  • Valores de comprimento: Você pode especificar a posição com valores de pixels ou porcentagem, como object-position: 20px 10px; para mover o conteúdo 20 pixels à direita e 10 pixels para baixo.

  • Percentuais: A posição pode ser ajustada em percentuais, como object-position: 50% 50%;, o que significa que o centro do conteúdo será alinhado com o centro do contêiner.

Essa propriedade é útil quando o conteúdo é cortado ao usar object-fit: cover, permitindo que você ajuste a parte da imagem que será visível (HubSpot Blog) (SitePoint).
